# py_pipeline_performance
End-to-end data analytics pipeline that transforms raw RPA execution logs into actionable business intelligence.

## 📌 Overview
Many organizations deploy automation bots but struggle to measure their actual ROI and identify operational bottlenecks. This project bridges the gap between automation execution and business intelligence. It provides a robust pipeline to generate, process, and visualize RPA (Robotic Process Automation) performance metrics.

## 🛠️ Tech Stack
* **Data Generation & ETL:** Python (Pandas, NumPy)
* **Operational Dashboard:** Tableau
* **Executive & ROI Dashboard:** Power BI
